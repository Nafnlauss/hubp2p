7. /src/hooks/use-user.ts
   Tipo: React Hooks (Client)
   Tamanho: ~4.8 KB
   Descrição: Hooks para autenticação e gerenciamento de usuário
   
   HOOKS IMPLEMENTADOS:
   
   useUser() → UseUserReturn
   Retorna:
   - user: User | null (Supabase Auth User)
   - profile: UserProfile | null (Perfil do usuário)
   - loading: boolean (Estado de carregamento)
   - error: Error | null (Erros)
   - isAdmin: boolean (Se é administrador)
   - isVerified: boolean (Se é verificado)
   - refetch: () => Promise<void> (Recarregar dados)
   
   useIsAdmin() → boolean
   - Retorna true se usuário é admin
   
   useRequireAuth(redirectTo?: string) → boolean
   - Protege rotas que requerem autenticação
   - Redireciona para /login se não autenticado
   
   useRequireAdmin(redirectTo?: string) → boolean
   - Protege rotas de admin
   - Redireciona para / se não for admin
   
   Funcionalidades:
   - Subscrição a mudanças de auth
   - Fetch automático de perfil
   - Tratamento de erros
   - Cleanup de subscriptions
   - RLS-ready para Supabase
