                      COMANDOS ESSENCIAIS
================================================================================

npx shadcn@latest init -d

# Adicionar componentes
npx shadcn@latest add button form input dialog table

# Instalar dependências
npm install react-hook-form zod @hookform/resolvers sonner

# Desenvolver
npm run dev

# Build
npm run build
